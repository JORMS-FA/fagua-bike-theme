# Changelog

Todas las modificaciones notables de este proyecto se documentan en este archivo.

El formato está basado en [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/),
y este proyecto sigue [Semantic Versioning](https://semver.org/lang/es/).

## [1.0.1] — 2026-06-29

### Agregado
- Header rediseñado: logo FAGUA + sub-navegación con categorías principales
- Sección de categorías visuales con iconos SVG (bicicletas, componentes, ruedas, accesorios, indumentaria, repuestos)
- Sección de productos destacados con grid responsive (8 productos del WC)
- Sección de ofertas con 4 productos en descuento
- Sección de bicicletas destacadas
- Sección de marcas premium (8 marcas: SHIMANO, SRAM, GW, MAXXIS, FOX, CONTINENTAL, RACEFACE, SHIMANO 105)
- Sección strip de confianza (pagos, envíos, garantía, WhatsApp)
- Hero compacto (48vh) con imagen cinematográfica
- Banner de servicio técnico
- Sección de reseñas
- Integración con Instagram
- Footer premium con información de contacto
- Catálogo sembrado con productos reales (bicicletas, componentes, ruedas, indumentaria, repuestos)
- Mini cart drawer
- Búsqueda AJAX
- SVG logo FAGUA con patch en fill de la "A" (azul Fagua #034DFC)
- Favicon SVG y apple-touch-icon
- Localización: `assets/images/`

### Técnico
- Child theme de Storefront
- PHP 8.0+
- WordPress 6.0+
- WooCommerce 8.0+
- Validado en Desktop (1440), Tablet (768) y Mobile (390)
- Sin errores de consola en los 3 viewports
- CLS 0 / 0 / 0.001
- Sin scroll horizontal
